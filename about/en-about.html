SOAP (originally Simple Object Access Protocol) — is a protocol specification for exchanging structured information in the implementation of web services in computer networks..

Initially SOAP applies to remote procedure calling (RPC) in most cases

Now this protocol uses to exchanging of custom messages in XML format , not only for procedures calling.

SOAP can operate over any application layer protocol such as : SMTP, FTP, HTTP, HTTPS and etc. However its iteration with this protocols has certain features, which must be defined individually. Commonly SOAP uses over HTTP

SOAP is one of standards, in which established web-service technologies
